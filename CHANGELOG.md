# Changelog

All notable changes to AgentForge will be documented in this file.

## [0.1.0] - 2026-06-26

### Added
- Initial project scaffolding (Electron + React + TypeScript + Tailwind)
- FastAPI backend with REST + WebSocket APIs
- Visual pipeline editor (Forge) powered by React Flow
- Agent creation wizard with role/model/capability configuration
- 4 collaboration modes: Sequential, Parallel, Conditional, Iterative
- Live pipeline execution monitoring (LiveMonitor)
- Task execution timeline (TaskTimeline) with Gantt-style visualization
- 5 pipeline templates: Code Review, Research, Content Gen, Debate, QA Workflow
- Agent adapters for Claude Code, Codex CLI, OpenCode, Gemini CLI + Custom
- MCP (Model Context Protocol) server/client implementation
- A2A (Agent-to-Agent) protocol implementation
- Zustand state management with history stack (undo/redo)
- Auto-layout via dagre library
- Pipeline import/export (JSON)
- Project CRUD with localStorage persistence
- Responsive industrial-themed UI (slate-900 + amber-400)
- 5W1H analysis document
- Comprehensive documentation (README, ARCHITECTURE, USER_GUIDE, DEVELOPER, ROADMAP)
- Multi-language support infrastructure
